# 开启云开发匿名登录

## 必须操作！

管理后台需要使用**匿名登录**来调用云函数。

## 操作步骤

### 1. 访问腾讯云 CloudBase 控制台

打开浏览器，访问：
```
https://console.cloud.tencent.com/tcb
```

### 2. 登录

使用**微信扫码登录**（必须是小程序管理员账号）

### 3. 找到环境

在环境列表中找到：`cloud1-9gzf2w8c9c9b7b73`

点击进入

### 4. 开启匿名登录

1. 左侧菜单点击：**环境** → **登录授权**
2. 找到 **匿名登录** 选项
3. 点击开关，**开启匿名登录**
4. 确认开启

## 截图示例

登录授权页面应该类似：

```
登录授权
├─ 微信登录        [已开启]
├─ 匿名登录        [点击开启] ← 这里
├─ 用户名密码登录  [未开启]
└─ 自定义登录      [未开启]
```

## 完成确认

开启后，你会看到匿名登录显示为 **[已开启]** 状态。

---

## 下一步

开启匿名登录后，继续下面的步骤：

### 步骤 4：构建项目

```bash
cd admin
npm run build
```

### 步骤 5：上传到静态托管

在微信开发者工具中：
1. 点击**云开发**图标
2. 选择**静态网站托管**
3. 如果未开启，点击**开通**
4. 点击**上传文件**
5. 选择 `admin/dist` 目录下的**所有文件**
6. 上传

上传完成后，会得到一个访问地址，类似：
```
https://cloud1-9gzf2w8c9c9b7b73.tcloudbaseapp.com
```

访问这个地址，使用 admin/admin123 登录即可！
