# AI助手使用说明

## 功能概述

暖白助手是基于硅基流动GLM-4-9B-Chat大模型的智能健康助手，提供两大核心功能：

1. **健康咨询**：专业的血液肿瘤健康知识问答
2. **智能记录**：通过自然语言快速记录健康数据

## 部署步骤

### 1. 部署云函数

在微信开发者工具中：

1. 右键 `cloudfunctions/callSiliconFlowAI` 目录
2. 选择 **上传并部署（云端安装依赖）**
3. 等待部署完成

### 2. 配置API密钥

1. 访问 https://cloud.siliconflow.cn/account/ak 获取硅基流动API密钥
2. 在微信云开发控制台中：
   - 进入 **云函数** → **环境变量**
   - 添加环境变量：
     - 名称：`SILICONFLOW_API_KEY`
     - 值：你的API密钥

### 3. 创建数据库集合

在云开发数据库中创建集合：
- `aiChatHistory` - 用于存储AI对话历史

建议索引：
- 字段：`openid`
- 类型：单字段索引
- 排序：升序

## 使用方法

### 健康咨询模式

1. 点击顶部 **健康咨询** 标签
2. 输入您的健康问题，例如：
   - "白细胞低怎么办？"
   - "化疗后如何护理？"
   - "GVHD是什么？"
3. AI会提供专业、温暖的健康建议

**示例对话**：
```
用户：白细胞低于正常值怎么办？
AI：白细胞偏低需要引起重视，建议：
1. 饮食调理：多吃高蛋白食物（鸡蛋、鱼肉、豆制品）
2. 避免感染：注意个人卫生，避免去人多的地方
3. 规律作息：保证充足睡眠，增强免疫力
4. 遵医嘱用药：如果医生开了升白药物，请按时服用
5. 定期复查：密切监测血象变化

如果白细胞持续低于1.0，请及时就医。
```

### 智能记录模式

1. 点击顶部 **智能记录** 标签
2. 可以通过两种方式记录：

**方式一：使用快捷按钮**
- 点击 "记录血常规"、"记录肝功能" 等按钮
- 按照AI引导逐步输入数据

**方式二：直接输入**
- 直接输入检验数据，例如：
  - "今天血常规：白细胞3.2，血小板80，血红蛋白90"
  - "肝功能：ALT 45，AST 38"
  - "EBV-DNA 5000拷贝"

3. AI会自动识别并解析数据
4. 确认后数据会自动保存到您的健康档案

**示例记录**：
```
用户：今天血常规：白细胞3.2，血小板80，血红蛋白90
AI：我已经识别到您的血常规数据：

- 白细胞(WBC): 3.2 × 10⁹/L
- 血小板(PLT): 80 × 10⁹/L
- 血红蛋白(HGB): 90 g/L

请确认数据是否正确？我将为您保存到健康档案。

✅ 数据已成功保存到您的健康档案
```

## 支持的检验项目

### 血常规 (bloodTest)
- WBC - 白细胞
- PLT - 血小板
- HGB - 血红蛋白
- NEUT - 中性粒细胞

### 肝功能 (liverFunction)
- ALT - 谷丙转氨酶
- AST - 谷草转氨酶
- TBIL - 总胆红素
- DBIL - 直接胆红素
- ALB - 白蛋白
- GGT - γ-谷氨酰转肽酶
- ALP - 碱性磷酸酶

### 肾功能 (kidneyFunction)
- CR/Creatinine - 肌酐
- BUN/Urea - 尿素氮
- UA - 尿酸
- eGFR/GFR - 肾小球滤过率

### 病毒学
- EBV-DNA (ebv)
- CMV-DNA (cmv)
- PP65抗原

### 其他
- LDH - 乳酸脱氢酶 (ldh)

## 常见问题

### Q: AI回复速度慢？
A: 这是正常现象，大模型需要一定时间进行推理。通常5-10秒内会收到回复。

### Q: 数据没有自动保存？
A: 请确保：
1. 已选择当前档案（在"我的"页面中）
2. 输入的数据格式正确
3. AI已经正确识别并返回JSON格式的数据

### Q: API调用失败？
A: 请检查：
1. 云函数是否已正确部署
2. API密钥是否已正确配置
3. API密钥余额是否充足

### Q: 如何清空对话历史？
A: 点击右上角的 **删除** 图标即可清空当前对话。

## 注意事项

1. **AI建议仅供参考**：AI提供的健康建议不能替代专业医疗诊断，重要决策请咨询主治医生
2. **紧急情况立即就医**：如遇紧急情况（高烧、严重感染、大出血等），请立即就医
3. **数据准确性**：AI识别的数据可能存在误差，请在保存前仔细核对
4. **隐私保护**：对话记录仅保存在您的个人云数据库中，不会泄露给第三方
