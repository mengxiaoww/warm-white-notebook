# 后台管理系统部署操作指南

## 第一步：部署云函数（⚠️ 必须完成）

### 在微信开发者工具中操作：

1. 打开项目根目录中的微信小程序
2. 点击左侧 **"云开发"** 图标（云朵形状）
3. 选择 **"云函数"** 标签
4. 找到以下8个云函数目录，**逐个上传**：

   ```
   ✅ adminLogin (管理员登录)
   ✅ adminGetUsers (用户列表)
   ✅ adminGetStats (统计数据)
   ✅ adminGetRecords (记录查询)
   ✅ adminGetUserDetail (用户详情)
   ✅ adminGetFeedbacks (反馈列表)
   ✅ adminUpdateFeedback (更新反馈)
   ✅ adminManageBanners (轮播图管理)
   ```

5. 对每个函数：**右键 → "上传并部署：云端安装依赖"**
6. 等待所有函数显示"部署成功"

## 第二步：上传静态网站

### 在微信开发者工具中操作：

1. 云开发控制台中，选择 **"静态网站托管"** 标签
2. 点击 **"文件管理"**
3. **删除旧文件**（如果存在）
4. 点击 **"上传文件"** 或 **"上传文件夹"**
5. 选择 `admin/dist/` 目录下的**所有文件**上传
6. 确认上传完成

## 常见问题

**Q: 登录后仍提示FUNCTION_NOT_FOUND？**
A: 检查是否所有8个admin云函数都已上传部署成功

**Q: 页面刷新后404？**
A: 重新上传admin/dist目录，确保包含cloudbaserc.json文件

**Q: 数据显示为0？**
A: 正常现象，因为数据库是空的。添加真实用户数据后会正常显示
