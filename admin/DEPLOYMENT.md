# 后台管理系统部署指南

## 云函数部署

### 1. 部署新增的云函数

在微信开发者工具中，右键以下云函数目录，选择「上传并部署（云端安装依赖）」：

```
cloudfunctions/adminManageSettings/
```

### 2. 初始化管理员账号

首次使用需要在云数据库中手动创建管理员账号：

1. 打开微信开发者工具
2. 点击「云开发」控制台
3. 进入「数据库」
4. 创建集合 `admins`
5. 添加第一个管理员记录：

```json
{
  "username": "admin",
  "password": "your_password",
  "email": "admin@example.com",
  "role": "super",
  "enabled": true,
  "createTime": "2025-10-03T00:00:00.000Z",
  "lastLoginTime": null
}
```

### 3. 初始化日志集合

创建集合 `adminLogs` 用于记录操作日志（系统会自动写入）。

## 功能说明

### 数据管理
- **实时统计**：显示用户数、档案数、记录数、反馈数、数据库大小
- **数据导出**：导出用户数据和反馈数据为Excel文件
- **清理过期数据**：删除超过1年的历史记录

### 管理员管理
- 查看所有管理员账号
- 添加新管理员（普通/超级管理员）
- 启用/禁用管理员账号
- 删除管理员（超级管理员不可删除）

### 操作日志
- 记录所有关键操作（登录、数据导出、配置修改等）
- 支持按类型筛选（登录/数据操作/系统配置）
- 支持按日期范围筛选
- 分页查询

## 注意事项

1. **密码安全**：当前密码为明文存储，生产环境建议使用加密算法（如bcrypt）
2. **权限控制**：建议添加更细粒度的权限管理
3. **数据备份**：导出功能仅导出用户和反馈数据，完整备份需要从云开发控制台操作
4. **日志清理**：建议定期清理旧日志，避免占用过多存储空间
