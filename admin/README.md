# 暖白记事本管理后台

基于微信云开发的 Web 管理后台系统。

## 功能特性

- 📊 数据概览：用户统计、记录统计、活跃度分析
- 👥 用户管理：用户列表、详情查看、档案管理
- 📝 记录查询：查看所有健康记录数据
- 💬 反馈管理：用户反馈处理

## 技术栈

- **前端**：Vue 3 + Element Plus + Vue Router + Pinia
- **后端**：微信云开发云函数
- **数据库**：微信云开发数据库
- **部署**：云开发静态网站托管

## 快速开始

### 1. 云函数部署

在微信开发者工具中，依次部署以下云函数：

```
cloudfunctions/
├── adminLogin          # 管理员登录
├── adminGetUsers       # 获取用户列表
├── adminGetStats       # 获取统计数据
└── adminGetRecords     # 获取记录数据
```

右键每个云函数目录 → 上传并部署（云端安装依赖）

### 2. 初始化管理员账号

在云开发控制台创建 `admins` 集合，手动添加管理员记录：

```json
{
  "username": "admin",
  "password": "admin123",
  "role": "admin",
  "createTime": "2025-10-03T00:00:00.000Z"
}
```

**生产环境请务必修改密码并使用加密存储！**

### 3. 开启云开发匿名登录

在微信开发者工具中：
1. 云开发控制台 → 设置 → 登录方式
2. 开启**匿名登录**（管理后台使用匿名登录调用云函数）

### 4. 本地开发

```bash
cd admin
npm install
npm run dev
```

访问 http://localhost:3000，使用 admin/admin123 登录。

### 5. 生产部署（推荐）

#### 方式一：云开发静态网站托管（无需服务器）

```bash
cd admin
npm run build
```

在微信开发者工具中：
1. 云开发 → 静态网站托管 → 设置
2. 开启静态网站托管
3. 上传文件 → 选择 `dist` 目录下所有文件上传
4. 访问生成的域名即可使用

**优势**：
- ✅ 完全免费（免费额度足够）
- ✅ 无需额外服务器
- ✅ 自动HTTPS
- ✅ CDN加速

#### 方式二：其他静态托管服务

构建后的 `dist` 目录可部署到任意静态网站托管服务：
- Vercel
- Netlify
- 阿里云 OSS
- 腾讯云 COS

## 重要配置

### 修改云环境 ID

如果使用不同的云环境，需修改 [admin/src/api/cloud.js](admin/src/api/cloud.js:4)：

```javascript
const ENV_ID = 'cloud1-9gzf2w8c9c9b7b73' // 修改为你的环境 ID
```

### 认证方式

管理后台使用**云开发匿名登录** + **云函数密码验证**：
1. 前端通过云开发 Web SDK 匿名登录
2. 调用 adminLogin 云函数验证管理员账号密码
3. 验证通过后返回 token，后续请求携带 token

无需额外服务器，完全基于云开发。

## 安全建议

1. **修改默认密码**：部署前务必修改管理员密码
2. **密码加密**：使用 bcrypt 等算法加密存储密码
3. **权限校验**：云函数中增加管理员权限验证
4. **HTTPS**：生产环境必须使用 HTTPS
5. **访问控制**：限制管理后台的访问 IP
