4. 点击 **确定**
5. 等待上传完成

上传完成后，在静态托管页面可以看到：

## 📝 注意事项

### 文件列表

确保 `dist` 目录包含以下文件：
```
dist/
├── index.html
├── assets/
│   ├── *.js
│   └── *.css
└── favicon.ico (如果有)
```

### 如果上传后访问 404

可能是没有配置索引文档，需要：

1. 在静态托管 → 设置
2. 设置 **索引文档** 为 `index.html`
3. 设置 **错误文档** 为 `index.html`（用于 Vue Router 的 history 模式）
